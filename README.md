# 매일약국

> 내 주변 병원, 약국 찾기 서비스
> 
> JSP와 SpringBoot를 이용한 공부 프로젝트

</br>

- 2022년 07월 12일 ~
- 총 5명

## :two: 사용기술
#### `Back-end`
- Java 8
- SpringBoot 2.6.0
- Gradle
- lombok 6.4.1
- 공공데이터

#### `Front-end`
- JavaScript
- JQuery
- Vue.js
- CSS3

## :four: 핵심 기능
지역과 시간, 진료 과목을 이용한 검색 기능이 이 서비스의 핵심 기능입니다.
이용자는 내 주변 진료/영업 중인 병원/약국을 쉽게 찾을 수 있고, 병원/약국에 대한 리뷰와 평가를 남길 수 있습니다.
